## 1. 프로젝트 소개

> **몰라도 된다, 알고만 있으면 되니까!**

알고리즘 학습부터 스터디, 실전 정보 공유까지  
함께 성장하며 **즉각적인 피드백**을 받을 수 있는 **올인원 플랫폼**

### 🐨 프로젝트 개요

**ALGO**는 개발 취업 준비생을 위한 통합 학습 플랫폼으로,
개인별 학습부터 협업 스터디, 기업별 실전 후기까지 한곳에서 관리할 수 있도록 설계되었습니다.  
사용자는 AI 피드백을 통해 자신의 코드 품질을 향상시키고,  
실제 기업의 코딩 테스트와 면접 정보를 인증 기반으로 신뢰성 있게 공유할 수 있습니다.

## 6. 동료 평가

### 강지륜

| 평가자 | 내용 |
|--------|------|
| 김민지 | 민지님 덕분에 실제 서비스 운영이 가능할 정도로 완벽한 UI 설계를 할 수 있었습니다. 이전까지는 단순 UI 설계만 해왔는데 민지님과 함께하면서 피그마 툴을 배울 수 있었습니다. 또한 백엔드 때와 마찬가지로 꼼꼼한 코드 리뷰를 통해서도 많은 부분을 배웠습니다. 시간이 부족한 상황에서도 발표 당일에 관리자 페이지까지 만들어주셔서 프로젝트 완성도를 높일 수 있었습니다. 또 프로젝트 시작할 때 공통 컴포넌트를 만들자고 제안해주셔서 개발 시간을 크게 단축할 수 있었고 디자인 통일할 수 있어서 정말 감사했습니다! |
